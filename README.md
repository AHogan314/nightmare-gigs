# nightmare-gigs
Craigslist Gigs Screen Scraper

## Installation
Copy the gigs .js file wherever you keep your node scripts, then: 
`npm install nightmare`
`npm install jQuery`


## Usage
`node gigs city`
The city argument can use any city on craigslist
